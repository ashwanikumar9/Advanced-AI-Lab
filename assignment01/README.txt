COMPILATION:
Open terminal or command prompt in the directory containing the files.

  Command to compile: g++ assg01.cpp -o assg01

EXECUTION:
The program takes two command-line arguments: 
1. the input filename and the 
2. target number of days (m).

Syntax:
   <executable_name> <input_filename> <number_of_days>

EXAMPLE:
   To run with Input File 1 with m = 4:
   for Linux/Mac: ./assg01 input01.txt 4
   for Windows:   assg01.exe input01.txt 4
